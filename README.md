# Basketball Scoreboard

An interactive basketball scoreboard web application featuring real-time score tracking, foul counters, and a game timer.

## Description

A fully functional basketball scoreboard with:
- Score tracking for Home and Guest teams (+1, +2, +3 points)
- Foul counter for each team
- Countdown timer (12 minutes per period)
- Period counter (up to 4 periods)
- Start/Pause controls
- New Game reset functionality

## Technologies

- HTML5
- CSS3
- JavaScript
- Vite (development environment)

## JavaScript Techniques Practiced

- DOM manipulation (getElementById, textContent)
- setInterval for timer functionality
- Conditional logic (if/else)
- Functions with parameters
- State management with variables
- Event handlers (onclick)

## Getting Started

```bash
npm install
npm run dev
```

## Game Rules

- Each game consists of 4 periods
- Each period lasts 12 minutes
- Timer automatically resets for each new period
- Game ends after the 4th period
